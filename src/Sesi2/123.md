## 1. Pengertian Class dan Object
Class adalah blueprint atau cetakan yang digunakan untuk membuat objek. Class berisi atribut (variabel) dan method (fungsi) yang mendefinisikan perilaku suatu objek.

Contoh:
```Java
class Mahasiswa {
    String nama;
    int umur;

    void belajar() {
        System.out.println("Mahasiswa sedang belajar");
    }
}
```

Object adalah hasil instansiasi dari class. Object merupakan representasi nyata dari class yang dapat digunakan untuk menjalankan method dan menyimpan data.

Contoh:
```Java
public class Main {
    public static void main(String[] args) {
        Mahasiswa mhs = new Mahasiswa(); // object
        mhs.nama = "Yasef";
        mhs.belajar();
    }
}
```

## 2. Jenis-Jenis Method
**1. Method Tanpa Return Value (void)**
Method yang tidak mengembalikan nilai.
```Java
void tampilkan() {
    System.out.println("Hello");
}
```

**2. Method dengan Return Value**
Method yang mengembalikan nilai.
```Java
int tambah(int a, int b) {
    return a + b;
}
```

**3. Method dengan Parameter**
Method yang menerima input.
```Java
void sapa(String nama) {
    System.out.println("Halo " + nama);
}
```

**4. Method Tanpa Parameter**
Tidak menerima input.
```Java
void halo() {
    System.out.println("Halo Dunia");
}
```

**5. Method Static**
Method yang bisa dipanggil tanpa membuat object.
```Java
static void info() {
    System.out.println("Ini method static");
}
```

## 3. Penjelasan Bagian pada Class (Sesuai Nomor)
Berdasarkan gambar:

**1. public class Komputer**
Ini adalah class declaration
- Menyatakan nama class yaitu Komputer
- public artinya bisa diakses dari mana saja

**2. String jenis_komputer; private String merk;**
Ini adalah atribut / variabel
- jenis_komputer → tipe komputer
- merk → merk komputer
- private → hanya bisa diakses di dalam class

**3. setDataKomputer(...)**
Ini adalah method setter
- Digunakan untuk mengisi nilai atribut
- this.jenis_komputer → menunjuk ke variabel dalam class

**4. getJenis()**
Ini adalah method getter
- Mengembalikan nilai jenis_komputer

**5. getMerk()**
Ini juga method getter
- Mengembalikan nilai merk

**6. Komputer mykom = new Komputer();**
Ini adalah pembuatan object (instansiasi)
- mykom adalah object dari class Komputer

**7. mykom.setDataKomputer(...)**
Pemanggilan method setter
- Mengisi data object

**8. System.out.println(...)**
Menampilkan output
- Menampilkan hasil dari getter
